Questions 1

  1. A server is software running on a computer that responds to requests made by clients. A web server stores web pages and sends them to clients on request.
  2. The client is a piece of software that connects to a server and displays the returned data in a meaningful manner. A browser is a client that connects to a web server, requests a file/files and renders the data returned as a web page.
  3. An IP address is the address of a device connected to a network. This address is used to route packets to their destination.
  4. A DNS (Domain Name Server) is a server which acts as a phone book for IP addresses. They map strings of text to IP addresses. Strings of texts are much easier to remember than strings of numbers.
  5. The Internet refers to the network connecting all computers and devices. The World Wide Web is an application that runs on Internet.
  6. HTML (HyperText Markup Language) describes the structure for web pages. It contains elements organized in a hierarchy where all of the elements have a parent element. The exception to this is the html-element which is the root element.

Questions 2

  1. The inline styling has a higher specificity than the external sheet.
  2. IDs are reserved for unique elements, elements that appear only once on the page. This could be the main header, or the menu bar etc. Classes are not unique, and can be used on multiple elements. Examples are info boxes, quotes, spans with specific formatting etc.
  3. RGBA (Red, Green, Blue, Alpha) is color coding including alpha. This adds an additional channel for transparency compared to RGB.
  4. Elements that occur within the body element are to be rendered on the page. Elements in the header element is is not rendered and is reserved for metadata, includes etc.
